Versions
Firmware version shows the current version of the XPort firmware and the build date (YYYYMMDD).
Web UI version shows the current version of the Web User Interface and the build date.
PIC Version shows the current versions of the embedded PIC microcontroller.
Setup version shows the current version of the setup table in the EEPROM.

Update

Download an available update package and read the file "ReadMe.txt" for instructions on how to use.
