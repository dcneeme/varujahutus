Help

Pullup
For each input switch the internal pull-up resistor (10 KOhms) on or off (no pullup)

Polarity
For each input select active state polarity e.g.:
"Low Act" to obtain an activate state when an attached button is pressed (Pullup=ON, button to connected between input and ground)
"Low Act" to obtain an activate state when an attached external power source(Pullup=OFF) is missing.
"Hi Act" to obtain an activate state when an attached external power source(Pullup=OFF) is present (>1.2Volt).

I/O Protocol
Choose between Dallas 1-Wire® on J5 (default setting) and Wigand® protocol interface on J6 connector.
Selecting Wiegand disables the Dallas 1-Wire temperature sensors and displays 0.0 °C for all 8 sensors on the status page (HOME, index.html).
